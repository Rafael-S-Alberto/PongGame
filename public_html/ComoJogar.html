Objetivo:

O objetivo do jogo Pong é simples: cada jogador controla uma raquete e tenta rebater a bola para o campo do adversário. A pontuação é feita quando um jogador não consegue rebater a bola, permitindo que ela passe pela raquete. O jogo continua até que um dos jogadores alcance o número de pontos necessário para vencer.

Controles Player 1:

Para mover o Player 1 no jogo Pong, utilize as teclas 'W' e 'S' do teclado. Pressione 'W' para mover a raquete do Player 1 para cima e 'S' para movê-la para baixo. Isso permite que o jogador posicione a raquete de forma a rebater a bola.

Controles Player 2:

Para mover o Player 2 no jogo Pong, utilize as teclas direcionais do teclado. Pressione a seta para cima para mover a raquete do Player 2 para cima e a seta para baixo para movê-la para baixo. Dessa forma, o jogador pode ajustar a posição da raquete e tentar rebater a bola.
